### Mirrorsize India — Deep Dive

- Founded ~2018, New Jersey-based, India dev team
- **App Store India ratings: 12** (twelve — not twelve thousand)
- Customers: GetStitched, Ira Soleli, Corpqlo, Samshek (all SMEs)
- 2019 claims: "India's biggest e-commerce retailer signed up" (unnamed), "brand with 20,000 stores going live" (unnamed)
- No consumer brand recognition after 7 years
- Pure B2B — invisible to end consumers (SDK embedded in other apps)
- **Verdict: Struggling B2B startup with negligible India traction after 7 years**

### Why Every Body Scanning App Has Struggled in India

1. Software-only, no physical presence
2. Invisible to consumers (embedded SDK, not an owned product)
3. No owned data relationship — data stays with the client brand
4. No repeat-use hook — one-size measurement is a one-time action
5. B2B sales cycle in fashion is long and relationship-driven

## The Uncontested Gap

Nobody in India has:
- Body measurement kiosk at mall/gym entrance
- Mall analytics + body scanning + fever screening combined
- Physical kiosk infrastructure network across Indian malls
- India-specific, city-segmented, consented face+body profile dataset
- Gym management software with native body scanning

## Competitive Moat Analysis

| Layer | Moat Strength | Why |
|---|---|---|
| App alone | None | Myntra/Nykaa clones in 6 weeks |
| Gym SaaS + scanning | Moderate | Differentiated but technically copyable |
| Physical kiosk network | Strong | Cannot be un-installed once in malls |
| Consented body profile dataset | Strongest | Compounding asset — 7-year head start possible |
| Fashion identity API | Platform moat | Infrastructure others build on top of |

---

## Key Validation Milestones (Before Kiosk Investment)

Before spending ₹1 crore on kiosks, validate three things:

1. **10 consented profiles** → proves the scanning tech works on real bodies
2. **1 gym owner pays anything** → proves willingness to pay
3. **1 fashion brand LOI for data** → proves data has commercial value

None of these require hardware. All three can be validated with the PWA MVP.
